ℹ️ О магазине

📚 Тема лабораторной работы

Интернет-магазин спортивных товаров — веб-приложение, разработанное в рамках лабораторной работы по курсу веб-разработки с использованием фреймворка Django (Python).

Цель проекта

Создание полноценного интернет-магазина с каталогом товаров, системой корзины, фильтрацией и поиском. Проект демонстрирует работу с моделями Django, представлениями, шаблонами и системой аутентификации.

🏪 Основан на магазине

Проект создан по образцу реального интернет-магазина спортивных товаров:

🌐 zonasporta.by

💻 Технологии

  • ✨ Backend: Django 5.0 (Python)
  • ✨ Frontend: HTML, CSS (встроенные стили)
  • ✨ Шаблоны: Django Templates
