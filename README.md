# Веб-приложение "Магазин спортивных товаров и питания" 

## Описание
Данное приложение является учебным проектом, созданным в рамках лабораторной работы по фреймворку Django.
Приложение реализует базовую структуру веб-сайта интернет-магазина.

**Тема проекта:** Разработка интерфейса для магазина спортивных товаров и питания

## Структура страниц

Проект содержит три основные страницы:

1. **Главная страница** (`/`)
   - Приветственное сообщение.
   - Навигационные ссылки на другие разделы сайта.

2. **Страница об авторе** (`/author/`)
   - Информация о студенте, выполнившем работу (ФИО, группа).

3. **Страница о магазине** (`/about/`)
   - Описание тематики магазина.
   - Краткое описание функционала, который планируется реализовать.
